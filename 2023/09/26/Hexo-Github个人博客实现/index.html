Hexo+github搭建个人博客

1.Node.js的安装及配置

参考原文：https://blog.csdn.net/weixin_52799373/article/details/123840137

1.1下载node.js

下载地址：https://nodejs.org/en/

1.2安装node.js

1.2.1程序安装

基本都可以next，一般只会更改安装位置。

1.2.2查看版本

node -v 查看node的版本

npm -v查看npm的版本（一般下载最新的node会自带最新的npm）

1.3配置环境变量

1.3.1配置全局安装的模块路径和缓存路径

1.3.1.1创建文件夹根目录

在node.js根目录，创建node_global,node_cache文件夹

管理员身份运行CMD，配置路径：

npm config set prefix "E:\develop\nodejs\node_global"
npm config set cache "E:\develop\nodejs\node_cache"

注意路径要匹配

1.3.2配置环境变量

win10&win11：右键此电脑——属性——高级系统设置——高级——环境变量

win11还可以：点设置——系统——关于——高级系统设置——高级——环境变量

创建NODE_HOME变量

修改系统变量中Path

添加node_global，node_cache：

1.4.测试

全局安装最常用的 express 模块 进行测试
命令如下:

npm install express -g

报错图片：

解决方法：删除npmrc文件（注意是C:\Users\（用户名）\这里的.npmrc）

1.5.配置淘宝镜像

npm config set registry https://registry.npm.taobao.org

安装cnpm

安装 cpm （按需安装）

npm install -g cnpm

查看配置

npm config ls

1.6.安装全局webpack

npm install webpack -g

2.Git的安装及配置

参考原文：https://www.cnblogs.com/xueweisuoyong/p/11914045.html

1.1下载Git

下载地址：https://git-scm.com/downloads

1.2安装Git

第二部可以更改安装位置 ，其他Next即可。

1.3生成SSH Keys

ssh-keygen -t rsa -C "你的邮箱地址"

找到密钥位置（id_rsa.pub文件）这里用记事本打开即可，复制密钥

Github绑定密钥

原文参考：https://blog.csdn.net/qq_38819293/article/details/122636647

测试是否绑定成功

ssh -T git@github.com

若询问yes or no 这里输入yes就行

1.4本地访问博客

1.4.1创建Blog文件，并在此启动Git Bash Here

1.4.2初始化hexo

hexo init

1.4.3生成本地hexo页面

hexo s

1.4.4本地访问

本地访问网址：http://localhost:4000/

在Git Bash里按 Ctrl+c可以关闭本地访问

1.5上传到Github

修改_config.yml文件（这里可以用记事本打开）

最下面

修改为

deploy:
type: git
repository: 你的github地址
branch: main

1.5.1安装hexo-deployer-git 自动部署发布工具

npm install hexo-deployer-git --save

生成界面

hexo g

若有此报错

就是_config.yml文件里修改时没加空格

1.5.2部署到Github

hexo d

中间会出现一个登录界面，可以用令牌登录。（令牌及时保存，就看不到了）

结束以后就上传 Github 就成功了！！！

如果出现错误重复尝试 直达报错消失
